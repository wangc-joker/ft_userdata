# 策略总览

这份文档用于快速说明当前目录里几条主要策略分别是做什么的、对应哪些交易对、以及目前回测的大致表现。

说明：

- 下面列出的收益、回撤、Sharpe，默认都是当前统一口径下的整段回测结果。
- 主要时间区间为 `2021-04-17` 到 `2026-04-09`。
- 主要模式为 `Isolated Futures`。
- 默认开启 `protections`。
